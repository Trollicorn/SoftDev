Introduction

If you haven't already, Bootstrap can be added to your page using a Content Delivery Network.
To add Bootstrap using a CDN, simply copy this line into the head of your page: 

<link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.1.3/css/bootstrap.min.css" integrity="sha384-MCw98/SFnGE8fJT3GXwEOngsV7Zt27NXFoaoApmYm81iuXoPkFOJwJ8ERdknLPMO" crossorigin="anonymous">

An alternative to using a CDN would be to download and use a local version of the CSS sheet, but this may cause more strain on your server.
If you want to learn more about this method, click here.

Containers

You can organize your page using Bootstrap's grid system. The grid system uses containers, rows, and columns to organize content. Sample code for a container with a row and a column looks like this: 

<div class="container">
  <div class = "row">
   <div class = "col-sm">
     The "col-sm" div creates a column. The number of columns in a row is determined by the number of "col-sm" divs.
   </div>
 </div>
</div>

Note: The "sm" in the column refers to the minimum width of the device viewing the page. Sizes include:

  • "sm": Small (576px)
  • "md": Medium (768px)
  • "lg": Large (992px)
  • "xl": Extra large (1200px)
  • A minimum size is not needed and using only "col" works for all sizes
There can be a maximum of 12 columns per row, but columns can be placed inside of columns.
If the size of a column is not specified (ex: <div class="col"></div>) then the column will fill all of the avaliable space.
If there are multiple columns without a specified size in a row, the width of the page will be split equally among them.

Buttons

Nice looking buttons can be added using Bootstrap. There are many different styles, and a button is created using this code: 

<button type="button" class="btn btn-XXXX">Button</button>

Here are the different styles of buttons available to you. To use a style, simply replace the XXXX in the code above with the style's name.



Keep in mind that class names are case-sensitive.

Other button styles

Note: All of these can be used in conjunction with any of the other above styles.

Outline buttons
<button type="button" class="btn btn-outline-primary">Button</button>


Disabled buttons
<button type="button" class="btn btn-secondary" disabled>Button</button>

The button is now disabled, but it's still clickable if someone is being sneaky and edits the HTML through their browser.



Classy buttons
<a style="color:white" class="btn btn-primary">Button</a>
<input class="btn btn-primary" type="submit" value="Submit">

Media

Bootstrap can display media objects alongside text.

Media heading
Text is formatted next to the image automatically.

Here is the code for media formatting: 

<div class="media">
    <img class="mr-3" style="width:10%;height:10%;"src="https://assets-cdn.github.com/images/modules/logos_page/GitHub-Mark.png" alt="git octopus">
    <div class="media-body">
        <h5 class="mt-0">Media heading</h5>
        Text is formatted next to the image automatically.
    </div>
</div>

Tables

Bootstrap tables are opt-in and are easy to customize.

col1 col2 col3 col4
1 1.1 1.2 1.3
2 2.1 2.2 2.3
3 3.1 3.2 3.3 
<table class="table">
  <thead>
    <tr>
      <th scope="col">col1</th>
      <th scope="col">col2</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th scope="row">1</th>
      <td>1.1</td>
    </tr>
  </tbody>
</table>
Table Customization

To make a table have light text on a dark background, create the table using

<table class="table table-dark">
...
</table>

Tables can be striped (rows alternate colors) by giving the table the "table-striped" class

Tables can have borders around the table and the cells by giving the table the "table-bordered" class

The table head can also be light or dark by giving the thead element the "thead-light" or "thead-dark" class

Other customization options can be found in the Bootstrap table documentation located here
